Exercise#7: Class creation
-------------------------------------------------------------------------------------------------

Creating a class, part 1:

syntactically the same as declaring a function

difference is that first letter is uppercase:

var Bike = function() {

                                      console.log("...calling the class Bike...");

                                   }



a function called bike might look this:

var bike = function() {

                                      console.log("...calling the function bike()...");

                                  }



bike();  // calling the function
Bike();  // this is syntactically similar to calling bike(). The difference is purely a semantic one and you shouldn't do it!

The reason for this semantic similarity is that you do not create classes in Javascript, but rather you create a constructor. The fact that the syntax of a class declaration (as shown above) should be the same as for a function declaration makes sense by virtue of the fact that a constructor is a form of function. -------------------------------------------------------------------------------------------------

Creating a class, part 2:

use of keyword new

Apart from defining the constructor for the class (part 1), class creation should also include the declaration of the variable for the new class.
This is achieved by use of the keyword 'new':

var bike = new Bike();

This does 3 things:
1. allocates memory for the instance bike
2. it calls the call() method - Bike.call(bike); - hence passing bike as the context object to Bike's constructor
3. assigns to the prototype: car.__proto__ = Car.prototype;

For the purpose of demonstration only, you can explicitly execute no.2 to show the context object working:

var Bike = function(model) {

                                               console.log("...calling with parameter...");
                                               this.model = model;                                    

                                            }

var example = { model : ""};
console.log(example);               // gives { model: "" }
Bike.call(example, "Carrera");
console.log(example);               // gives { model: "Carrera" }

Creating a class, part 3:

use of keyword new, with properties

var Bike = function(model) {

                                               console.log("...calling with parameter...");
                                               this.model = model;                                    

                                            }

var bike = new Bike("Cervelo");
console.log(bike);               // gives { model: "Cervelo" }
