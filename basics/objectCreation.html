Exercise#6: Object creation
-------------------------------------------------------------------------------------------------

Creating example object: a person with name and age attributes


var person = {

                         age : 54,
                         name : "Brendan Eich"

                     };


console.log(person.age);    // gives the age
console.log(person.name);    // gives the name
person.playsFootball());    // calls person's function playsFootball()
