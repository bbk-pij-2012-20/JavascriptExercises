Exercise#6: Object creation
-------------------------------------------------------------------------------------------------

Creating example object: a person with attributes (name & age) and behaviour (writing javascript)


var person = {

           age : 54,
           name : "Brendan Eich"

           writeJavascript : function() {

                     console.log("...writing Javascript...");

           }

};


console.log(person.age);       // gives person's age
console.log(person.name);    // gives person's name
person.writeJavascript());    // calls Brendan's behaviour - writeJavascript()
