Exercise#3: Type coercion
-------------------------------------------------------------------------------------------------

double equals, ==, performs type conversion (prior to comparison)

console.log(1 == 1.0);    // gives true

console.log("1" == 1);    // gives true

console.log(1 == "1.0");    // gives true

console.log("1" == "1.0");    // gives false

-------------------------------------------------------------------------------------------------

triple equals, ===, compares types as well as values. Always aim to use this instead of == to avoid generating bugs.

console.log(1 === 1.0);    // gives true. Note that Javascript only has one type for all numbers, namely a 64-bit Floating Point number.
                                          // (It is not like Java which has short, long, integer, float and double).
console.log("1" === 1);    // gives false

console.log(1 === "1.0");    // gives false

console.log("1" === "1.0");    // gives false
