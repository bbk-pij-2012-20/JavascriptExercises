Exercise#3: Type coercion
-------------------------------------------------------------------------------------------------
console.log("1" == 1);// gives true

console.log(1 == "1.0");// gives true

console.log("1" == "1.0");// gives false

// Problem with this is that the double equals, ==, performs type conversion (prior to comparison) -------------------------------------------------------------------------------------------------
console.log("1" === 1);// gives false

console.log(1 === "1.0");// gives false

console.log("1" === "1.0");// gives false

// Triple equals, ===, compares types as well as values. Always aim to use this instead of == to avoid generating bugs.
