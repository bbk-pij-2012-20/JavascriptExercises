functions exercise#2: arguments & parameters
-------------------------------------------------------------------------------------------------

numbers of arguments & parameters, part#1

A function biggerNumber is defined that takes two parameters (named param1 and param2).
Some operation is carried out using these two parameters and returns a value accordingly.
If you call the function and pass only 1 parameter, the result is undefined (because param2 is undefined).
If you call the function and pass more than 2 parameters, the function still just uses the first two parameters (named param1 and param2).
However, the function can access any parameter passed to it using the object 'argument' (see part#2).

var getMax = function(param1, param2){

          if (param1 > param2) {

               return param1;

          } else {

               return param2;

          }

};

console.log(getMax(10));               // gives undefined
console.log(getMax(10, 20));         // gives 20, which is correct
console.log(getMax(10, 20, 30));  // gives 20, which is incorrect

numbers of arguments & parameters, part#2

The function can access any parameter passed to it using the object 'arguments':

var getMax2 = function(){

          var max = arguments[0];

          for (var i = 1; i < arguments.length; i++) {

               if (max < arguments[i]) {

                    max = arguments[i];

               }

          }

          return max;

};

console.log(getMax2(10));                                            // gives undefined
console.log(getMax2(10, 20));                                      // gives 20, which is correct
console.log(getMax2(10, 20, 30, 5, 56, 234, 2, 120));  // gives 234, which is correct
