functions exercise5: closures
-------------------------------------------------------------------------------------------------
$(document).ready(function() {

       var n = 1;
       $('button').on('click', function() {

       n++;              alert(n);

       });

});

the variable n is declared in a different scope from the event listener's alert
Javascript notices that there is a reference made to n in the other scope.
As a result, Javascript saves the variable n in a section of memory called a closure.
Now the alert(n) and the n++ will continue to be able to access the variable albeit from
a different scope
