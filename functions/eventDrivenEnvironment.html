functions exercise#x: event-driven environment
-------------------------------------------------------------------------------------------------
var a = 1;
console.log("this message won't get run again and again");

$(document).ready(function() {

       $('button').on('click', function() {

             alert(a);

       });

});

The console log message is only produced once, not every time the button is clicked.
Only the event-listener in the function is run each time the button is clicked.
This is what's meant by javascript running in an event-driven environment.
Unlike PHP for example, the script is stored in memory and this is why Javascript execution is fast.
