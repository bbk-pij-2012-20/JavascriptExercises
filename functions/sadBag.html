In the client-side (i.e. not using node.js), importing functions from external js files is shown here via the browser without use of any front-end js frameworks:
Achieved simply by referencing the js file with the src attribute in the script tags.
Note that the script element, with the src attribute specified, ignores any text between its start and end tags.
The function call needs to be wrapped inside a second, subsequent script element that has no src attribute.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------

<body/>

    <script src="happyFun.js"></script>

    <script>

        haveFun();
        getHappy();

    </script>

</body>
